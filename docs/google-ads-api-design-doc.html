Note: this tool is externally accessible, so screenshots/mockups of the tool are included below.

Company Name: Quantana Pty Ltd

Business Model: Quantana is a technology and AI consulting company (quantana.com.au). We work with a range of small and medium business clients across Australia and internationally. We run Google Ads campaigns for our own AI services as well as manage campaigns on behalf of our consulting clients. Our tool "Babji" is an AI business assistant that our clients use via Telegram and WhatsApp to manage their day-to-day business operations, including monitoring and managing their Google Ads campaigns through natural language conversation.

Tool Access/Use: Babji is used by both internal employees and external clients. Our consulting clients connect their own Google Ads accounts to Babji via OAuth, and then interact with it through Telegram or WhatsApp chat. They can ask questions like "how are my ads doing this month?" and Babji will pull the relevant data from the Google Ads API and summarize it conversationally. Internal staff at Quantana also use Babji to monitor client campaigns and spot issues. Each user's Google Ads data is completely isolated — one client cannot see another client's data. OAuth tokens are encrypted at rest using AES-256-GCM.

Tool Design: Babji uses a conversational AI (ReAct loop) that interprets user messages and decides which Google Ads API calls to make. For reporting, it queries the Google Ads API in real-time using the searchStream endpoint with GAQL queries, then the AI summarizes the metrics into plain language for the user. We do not store Google Ads data in our own database — it is fetched on-demand, summarized, and presented in chat. For campaign management, users can update budgets and pause/enable campaigns through conversation, with confirmation required before any changes are applied.

API Services Called:

  • Pull campaign, ad group, and keyword performance reports using the Campaign, AdGroup, and KeywordView resources via GoogleAds.SearchStream
  • Pull audience demographic data using the GenderView, AgeRangeView, and device segments
  • Update campaign budgets via the CampaignBudgetService mutate endpoint
  • Pause and enable campaigns via the CampaignService mutate endpoint
